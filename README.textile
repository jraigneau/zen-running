h1. ZenRunnin'

Zenrunnin' est une application de suivi de performances de vos courses à pied, utilisant votre compte facebook pour partager avec vos amis

h2. Introduction

Zenrunnin' est une application simple de suivi de vos courses, utilisant votre compte facebook pour partager vos performances avec vos amis et connaître les leurs.

Le site Zenrunnin' est à la base un petit projet pour jouer avec les API facebook et mieux les connaître - il n'a pas vocation à devenir payant et reste sans garantie
