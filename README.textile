h2. Introduction

"Zenrunnin'":http://zenrunn.in/ est une application simple de suivi de vos courses, utilisant votre compte facebook pour partager vos performances avec vos amis et connaître les leurs.

Le site "Zenrunnin'":http://zenrunn.in/ est à la base un petit projet pour jouer avec les API facebook et mieux les connaître - il n'a pas vocation à devenir payant et bien que sérieusement testé et validé, il reste sans garantie sur sa pérénité ou sur les anomalies.

Pour les geeks: "Zenrunnin'":http://zenrunn.in/ est écrit en "Ruby":http://www.ruby-lang.org/fr/, utilise le framework "Sinatra":"http://www.sinatrarb.com/, et la librairie "Koala":https://github.com/arsduo/koala pour accéder à Facebook tout en étant hébergé sur "Heroku":http://www.heroku.com/.
